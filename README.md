# Chariot of Heaven

Take a look at [the project overview](project.pdf) to learn more.

### Commands

- `w` start moving forward
- `s` start moving backward

- `a` start turning left
- `d` start turning tight
- `z` stop turning

- `x` stop all

### Wiring

**Info diodes**

- red - GPIO17
- green - GPIO22
- yellow - GPIO27

**Servo**

- servo - GPIO10

\*_Stepper motor_
stepper motor driver: MP6500 Pololu 2966

- DIR - GPIO21
- STEP - GPIO20
